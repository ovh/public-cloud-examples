This tutorial explains how to route your requests (prompts) using [**LiteLLM**](https://www.litellm.ai/) when working with LLM as a Service (LLMaaS) such as OVHcloud [**AI Endpoints**](https://www.ovhcloud.com/en/public-cloud/ai-endpoints/).

### Prerequisites

- Python `3.13.x` installed
- create AI Endpoints API key following this [documentation](https://help.ovhcloud.com/csm/en-gb-public-cloud-ai-endpoints-getting-started?id=kb_article_view&sysparm_article=KB0065401#generating-your-first-api-access-key) to know how to generate it
- use any of the LLM instruct models
- define `OVHCLOUD_AI_ENDPOINTS_API_KEY` environment variable in your dev environment

### How to use the project

To follow this tutorial in the best conditions and to learn more about how to use OVHcloud AI Endpoints as a [model provider in LiteLLM](https://docs.litellm.ai/docs/providers/ovhcloud), please refer to this [**guide**](https://help.ovhcloud.com/csm/en-gb-public-cloud-ai-endpoints-litellm-integration?id=kb_article_view&sysparm_article=KB0072657).
